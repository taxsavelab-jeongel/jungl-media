# 인터넷신문 등록 가이드 — 정엘미디어

「신문 등의 진흥에 관한 법률」(신문법)에 따른 인터넷신문 등록 절차 정리입니다.
(2026년 7월 기준 공개 자료 기반 — 신청 전 관할 시·도 담당 부서에 최종 확인을 권합니다.)

## 왜 등록해야 하나

등록하면 언론사 지위를 갖게 되어 "신문", "기사", "기자" 명칭을 정식으로 사용할 수 있고,
포털 뉴스 제휴 신청, 언론중재위 절차 참여, 광고 영업에서 공신력이 생깁니다.
등록 없이 운영하면 매체가 아닌 일반 웹진·블로그 지위이므로 표현에 제약이 있습니다.

## 등록 요건 (핵심)

- **자체 기사 생산**: 주간 게재 기사의 30% 이상을 자체 생산 기사로 게재
- **주간 발행**: 주간 단위로 새로운 기사를 게재 (지속적 발행)
- **발행 주체**: 개인 또는 법인 모두 가능 — 정엘의 경우 연구소 법인 명의 등록을 검토
- **결격사유**: 발행인·편집인이 신문법상 결격사유(일정 전과 등)에 해당하지 않아야 함

## 등록 기관·절차

관할: **발행소 소재지 시·도지사** (서울이면 서울특별시청 담당 부서)

절차: 등록신청서 제출 → 구비서류 확인·접수 → 등록기준 심사 → 결격사유 조회 → 등록수리 통보 → **등록증 교부** (통상 7일~25일)

신청은 방문·우편 접수 또는 정부24 (검색: "신문, 인터넷신문 사업 등록") 로 가능합니다.

## 구비서류 (법인 발행 기준)

1. 인터넷신문사업 등록신청서 1부 (시·도 서식 또는 정부24)
2. 발행인·편집인 **기본증명서** 각 1부 (동일인이면 1부)
3. **법인 등기사항전부증명서** 1부
4. 발행소 사용 권원 증명 — 자가면 건물 등기부, **임차면 임대차계약서 사본** 1부
   (임차인 명의가 발행 주체와 일치해야 함)
5. 시·도에 따라 정관, 주주명부 등을 추가 요구할 수 있음 → 사전 전화 확인

## 등록신청서에 기재할 사항 (미리 결정해 둘 것)

| 항목 | 정엘미디어 (안) |
|---|---|
| 제호 | 정엘미디어 |
| 종별 | 인터넷신문 |
| 간별 | 주간 이상 수시 발행 |
| 발행 목적·내용 | 가업승계·세무·중소기업 경영 전문 보도 |
| 인터넷 주소 | media.연구소도메인 (배포 후 확정) |
| 발행인 / 편집인 | 결정 필요 (겸임 가능) |
| 발행소 | 연구소 주소 |

## 등록 후 의무

- **제호·등록번호 표시**: 사이트에 등록번호, 등록일, 발행인·편집인, 발행소 주소, 전화번호, 상호 표기 (사이트 푸터에 이미 반영됨 — `js/config.js`에 값만 입력)
- **청소년보호책임자** 지정·표시 (푸터 반영됨)
- 등록사항(발행인, 주소, 제호 등) 변경 시 **변경 등록**
- 개인정보처리방침 게시 (회원 기능 운영 시)

## 진행 순서 제안

1. 사이트 배포 완료 (임시 vercel.app 주소라도 가능) 및 초기 기사 5~10건 발행 — 심사 시 실제 운영 화면 확인 대비
2. 발행인·편집인·청소년보호책임자 확정, 서브도메인 연결
3. 관할 시·도 담당 부서 전화 확인 (추가 서류·서식)
4. 서류 접수 → 등록증 수령 → `js/config.js`에 등록번호 입력

※ 본 문서는 일반 정보 제공용이며 법률 자문이 아닙니다. 세부 요건은 관할 시·도 및 국가법령정보센터의 현행 법령을 기준으로 최종 확인하세요.
